## 2. Protocols Used Directly from the Terminal

| Protocol  | Terminal Command or Tool                |
|-----------|------------------------------------------|
| SSH       | `ssh user@ip`                            |
| FTP/SFTP  | `ftp ip`, `sftp user@ip`                |
| Telnet    | `telnet ip port`                         |
| HTTP(S)   | `curl`, `wget`                           |
| SMTP      | `telnet smtp.server.com 25`             |
| SNMP      | `snmpwalk`, `snmpget`                   |
| DNS       | `dig domain.com`, `nslookup domain.com` |
| LDAP      | `ldapsearch`                             |
| WebSocket | `wscat` (via `npm install -g wscat`)     |
| NFS/SMB   | `mount -t nfs`, `smbclient`              |

## 5. Best Protocols to Learn for the Future

| Protocol           | Why Important                            | Usage                                     |

| HTTP/2, HTTP/3     | Faster modern web                        | Web development, APIs                     |
